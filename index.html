Summary: Latitude vs. Weather Parameters


The purpose of the project was to analyze how the weather (namely maximum temperature, humidity, cloudiness and wind speed) varies as the latitude changes. To perform the analysis, about 1200 latitudes and longitudes were randomly generated. Then, citipy python library was used to gather at least 500 unique cities and OpenWeatherMap API was used to pull weather data on these cities.

After compiling the data, Matplotlib was used to plot the above mentioned weather attributes against latitude. This site provides the source data and plots as well as analysis of the data.
